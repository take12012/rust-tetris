## memo
|  キー  |  説明  |
| ---- | ---- |
|  `#[derive(Clone, Copy)]`  |  Outer attribute。直後のアイテムに属性を付与する。  |
|  &#124;=  |  ビットOR後に代入。オーバーロード「BitOrAssign」。  |
|  getch-rs = "0.1"  |  外部クレート。キー入力の取得。`cargo add getch-rs`コマンドでもOK。  |
|    |    |
